## Nasıl Kullanılır

1. Web sayfasını açın.
2. Klavyeden "A", "S", "D", "F", "G", "H", "J", "K" veya "L" tuşlarına basın.
3. İlgili harfe basıldığında, atanmış olan ses efekti çalınacaktır.
